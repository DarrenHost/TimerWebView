# TimerWebView v1.5 UI 设计说明

## 📱 设计图位置
- **文件**: `design/TimerWebView-v1.5-design.drawio`
- **格式**: draw.io (可导入在线编辑)
- **尺寸**: 414x896 (标准手机屏幕)

## 🎨 设计改进点

### 1. 状态栏 (Status Bar)
- **高度**: 44dp
- **背景**: 纯黑色 (#000000)
- **内容**: 时间、通知图标、5G 信号、电池

### 2. 应用标题栏 (App Header)
- **高度**: 60dp
- **背景**: Bootstrap Primary Blue (#0d6efd)
- **标题**: "TimerWebView" - 白色 20px 粗体
- **倒计时徽章**: 
  - 圆角矩形，半透明蓝色背景
  - 白色数字显示 (00:17)
  - 尺寸：90x30dp
- **功能按钮**:
  - 全屏按钮 (⛶) - 白色图标
  - 设置按钮 (⚙) - 白色图标

### 3. WebView 内容区
- **背景**: 纯白色 (#FFFFFF)
- **边框**: 浅灰色 (#dee2e6)
- **错误页面布局**:
  - Android 机器人图标 (64px)
  - 错误标题："网页无法打开" (22px 粗体)
  - 错误描述 (15px 灰色)
  - 错误代码框 (单色字体，红色)
  - **重试按钮**: Bootstrap Primary 风格
    - 蓝色背景 (#0d6efd)
    - 白色文字
    - 圆角 6dp
    - 尺寸：150x44dp

### 4. 底部倒计时浮层
- **位置**: 底部中央
- **背景**: 浅灰色 (#e9ecef)
- **文字**: "下次刷新：00:17" (13px 灰色)
- **尺寸**: 150x32dp
- **圆角**: 6dp

### 5. 设置页面 (右侧)
- **背景**: 浅灰色 (#f8f9fa)
- **卡片式布局**:

  - 浅灰边框

   - 点击刷新功能

### 设置界面功能

2. ✅ **刷新间隔**

3. ✅ **高级选项**
   - JavaScript 开关
   - 缓存开关
   - 恢复默认配置

4. ✅ **版本信息**

## 📱 交互说明

### 全屏模式
- 点击全屏按钮进入/退出
- 隐藏状态栏 + 导航栏 + 标题栏
- WebView 点击 5 次退出全屏

### 倒计时
- 标题栏徽章：仅显示时间 (MM:SS)
- 底部浮层：显示"下次刷新：MM:SS"
- 点击徽章可手动刷新

### 设置页面
- 点击设置按钮进入
- 修改后点击"保存配置"生效
- "恢复默认"重置所有配置
- 自动返回主页面
